# 🍔 Food Taka Tak - Online Food Ordering Website

Food Taka Tak is a modern and responsive food ordering website where users can browse food items, add them to the cart, and place orders easily.

## ✨ Features

* Home Page with attractive UI
* Food Categories
* Popular Food Items
* Add to Cart
* Search Food
* Responsive Design
* Contact Section
* Fast and User-Friendly Interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## 📁 Project Structure

```text
Food-Taka-Tak/
│── index.html
│── css/
│── js/
│── images/
│── about.html
│── contact.html
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enjoy the Food Taka Tak website.
